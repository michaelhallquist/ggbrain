# breaks function for including min + max with labels, and a few unlabeled ticks in between

breaks function for including min + max with labels, and a few unlabeled
ticks in between

## Usage

``` r
range_breaks(n = 3, digits = 2)
```

## Arguments

- n:

  number of breaks added within the min-max range

- digits:

  number of decimal places to display

## Value

a function for generating breaks on a continuous scale with the min and
max labeled
