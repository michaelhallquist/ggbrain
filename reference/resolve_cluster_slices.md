# Resolve a deferred cluster_slices_spec

Resolve a deferred cluster_slices_spec

## Usage

``` r
resolve_cluster_slices(spec, images, layers = NULL)
```

## Arguments

- spec:

  A cluster_slices_spec object

- images:

  A ggbrain_images object

- layers:

  A list of ggbrain_layer objects (optional, for layer reference)

## Value

A list with elements: coordinates (character vector) and cluster_data
(data.frame)
